# Customer Sales Analytics Dashboard

## Overview
This project analyzes sales performance across 100,000+ customer and transaction records to identify revenue drivers and customer behavior.

## Tools Used
SQL, Python, Power BI

## Dataset
Simulated e-commerce dataset with 100k+ records.

## What I Did
- Cleaned and structured the data using Python and SQL.
- Wrote advanced SQL queries (JOINs, GROUP BY, subqueries) to analyze sales trends, customer segments, and product performance.
- Built an interactive Power BI dashboard to visualize key metrics including revenue, customer growth, and top-selling products.
- Created interactive filters (region, product category, time) to support business decision-making.

## Key Results
- Identified products contributing to ~45% of total revenue.
- Improved reporting efficiency by 40% by automating data extraction and dashboard updates.
